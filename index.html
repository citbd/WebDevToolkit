Motion & Effects

✨ Animation Libraries

Add smooth animations and scroll effects to your website with these easy CDN libraries.

📜
AOS — Animate On Scroll
Animate elements as they scroll into view. Add CSS + JS to <head> and </body>.
CSS — paste in <head>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aos@2.3.4/dist/aos.css">
JS — before </body>
<script src="https://cdn.jsdelivr.net/npm/aos@2.3.4/dist/aos.js"></script> <script>AOS.init();</script>
Usage — add data-aos to any element:
HTML
<div data-aos="fade-up">Animates on scroll!</div> <div data-aos="zoom-in" data-aos-delay="200">Delayed zoom!</div>
View Demo →
💫
Animate.css
Just add a class to any element for instant CSS animations. No JS needed for basic use.
CSS — paste in <head>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/animate.css@4.1.1/animate.min.css">
Usage — add classes to any element:
HTML
<h1 class="animate__animated animate__fadeIn"> Fades in! </h1> <p class="animate__animated animate__bounceIn"> Bounces in! </p>
View All Animations →

Search Engine Optimization

SEO Meta Tags

Add these inside your <head> to help search engines understand your page.

🔎 Essential SEO Tags
Description, keywords, and author — the three must-haves.
HTML — paste in <head>
<meta name="description" content="Free Web tutorials"> <meta name="keywords" content="HTML, CSS, JavaScript"> <meta name="author" content="John Doe">
⚙️ HTTP-Equiv Meta Tags
Browser behavior, language, caching, and compatibility settings.
HTML — paste in <head>
<meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta http-equiv="content-language" content="en"> <meta http-equiv="cache-control" content="no-cache"> <meta http-equiv="refresh" content="30">
📱 Open Graph Tags (Social Share)
Control how your site looks when shared on Facebook, WhatsApp, etc.
HTML — paste in <head>
<meta property="og:title" content="My Website"> <meta property="og:description" content="Free Web tutorials"> <meta property="og:image" content="https://example.com/image.jpg"> <meta property="og:url" content="https://example.com"> <meta property="og:type" content="website">
🐦 Twitter Card Tags
Control how your site looks when shared on Twitter / X.
HTML — paste in <head>
<meta name="twitter:card" content="summary_large_image"> <meta name="twitter:title" content="My Website"> <meta name="twitter:description" content="Free Web tutorials"> <meta name="twitter:image" content="https://example.com/image.jpg">
🧩 All SEO Tags Combined — Copy Once
Paste this full block inside your <head> and customize the values.
HTML — Full SEO Block
<!-- Primary SEO --> <meta name="description" content="Free Web tutorials"> <meta name="keywords" content="HTML, CSS, JavaScript"> <meta name="author" content="John Doe"> <meta name="robots" content="index, follow"> <!-- HTTP-Equiv --> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta http-equiv="content-language" content="en"> <meta http-equiv="cache-control" content="no-cache"> <!-- Open Graph (Social) --> <meta property="og:title" content="My Website"> <meta property="og:description" content="Free Web tutorials"> <meta property="og:image" content="https://example.com/image.jpg"> <meta property="og:url" content="https://example.com"> <meta property="og:type" content="website"> <!-- Twitter Card --> <meta name="twitter:card" content="summary_large_image"> <meta name="twitter:title" content="My Website"> <meta name="twitter:description" content="Free Web tutorials"> <meta name="twitter:image" content="https://example.com/image.jpg">
